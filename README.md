# loan-approval-prediction
This project predicts loan approval using Machine Learning. It includes data preprocessing, EDA, and label encoding. Logistic Regression and Random Forest models are trained and compared. The project achieves high accuracy and includes visualizations, confusion matrix, feature importance, and model saving using pickle for future deployment
## Objectives
- Perform data cleaning and preprocessing
- Analyze loan applicant data
- Train and evaluate Machine Learning models
- Compare model performance
- Identify important features affecting loan approval

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Pickle

## Machine Learning Models
1. Logistic Regression
2. Random Forest Classifier

## Project Workflow
1. Load and explore the dataset
2. Handle and encode categorical variables
3. Perform Exploratory Data Analysis (EDA)
4. Split data into training and testing sets
5. Train Machine Learning models
6. Evaluate model performance
7. Visualize results
8. Save the trained model

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report
- Feature Importance Analysis

## Results
| Model | Accuracy |
|---------|----------|
| Logistic Regression | ~80% |
| Random Forest | ~98% |

Random Forest achieved the highest accuracy and was selected as the final model.
